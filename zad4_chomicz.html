JavaScript jest językiem obiektowym, ponieważ umożliwia tworzenie i manipulowanie obiektami, które mogą zawierać dane i funkcje.
Obiekty w JavaScript są kluczowym elementem programowania w tym języku, pozwalając na organizację kodu w sposób,
który odzwierciedla rzeczywiste byty i ich interakcje.
Dodatkowo, JavaScript wspiera dziedziczenie prototypowe, co jest istotnym aspektem obiektowości.
6)wymień zalety JS:
Wszechstronność: JavaScript działa zarówno po stronie klienta (w przeglądarkach), jak i serwera (dzięki Node.js), co pozwala na tworzenie pełnoprawnych aplikacji webowych.
Interaktywność: Dzięki JavaScript można tworzyć dynamiczne i interaktywne elementy na stronach internetowych, poprawiając doświadczenia użytkowników.
Szeroka społeczność i ekosystem: JavaScript ma ogromną społeczność i bogaty ekosystem bibliotek oraz frameworków, takich jak React, Angular czy Vue, co ułatwia rozwój aplikacji.

12)Opisz następującą instrukcję window.close()
Instrukcja window.close() w JavaScript służy do zamykania bieżącego okna przeglądarki. Może być używana tylko w określonych sytuacjach, takich jak:
Okna otwarte przez skrypt: Możesz zamknąć okno, które zostało otwarte przez skrypt (np. za pomocą window.open()).
Ograniczenia: Większość nowoczesnych przeglądarek nie pozwala na zamykanie głównego okna przeglądarki, które zostało otwarte przez użytkownika bezpośrednio, aby zapobiec nadużyciom.
